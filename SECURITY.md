## Reporting a vulnerability

If you believe you've found a security issue in **paperclip-adapter-hermes-gateway** (@felipefontoura/paperclip-adapter-hermes-gateway), please **do not open a public GitHub issue**.

Instead, email **ffontouras@gmail.com** with:

- A short description of the issue.
- Steps to reproduce, or a proof-of-concept.
- The version you tested against (npm version or container tag).
- Any suggested mitigation.

You will receive an acknowledgement within **5 business days** and a status update within **15 business days**. If the report is confirmed, we'll coordinate a fix and a coordinated disclosure window of up to **30 days** before publishing a release and an advisory.

Thank you for helping keep the ecosystem safe.
